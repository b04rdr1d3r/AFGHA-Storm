Afghanistan Storm
by Blue Storm (2026)

A template map for DCS world
emulating the Enduring Freedom operations

It provides:
populated airbases at Kandahar / bagram / Camp bastion and several FARPS
pre-determined zone of operations and actions (,orth, Saguin valley and Uzbin region ambush)
random zones where TIC may occur (MOOSE)
Random air Traffic (MOOSE)
CTLD and CSAR (using MIST, CTLD and CSAR scripts)
3 tanker tracks
an AWACS

requires MOOSE, MIST, CTLD and CSAR scripts (included)
All custom scripts are in the repository should you want to tweak them
